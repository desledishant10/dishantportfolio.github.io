- Smooth scrolling
- Animated on-scroll effects
- Projects section with descriptions and source code links
- Certifications section with viewable certificates
- Embedded resume with download option
- Contact form with form validation and email sending functionality

1. **Clone the repository**:
   ```bash

   cd dishantportfolio.github.io
2. Open index.html in your browser if you are not using a local server setup.
## Usage
- Navigate through the different sections of the portfolio using the navigation bar.
- View projects and their details in the Portfolio section.
- Check certifications in the Certifications section.
- View and download my resume in the Resume section.
- Contact me using the contact form in the Contact section.

## File Structure
- assets/: Contains all images and the resume PDF used in the portfolio.
- styles.css: Contains all the CSS styles for the portfolio.
- script.js: Contains JavaScript for AOS initialization, smooth scrolling, toggling the mobile menu, and form validation.
- send_email.php: PHP script to handle the form submission and send emails.
- index.html: The main HTML file for the portfolio.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- PHP (for form handling)
- [AOS (Animate On Scroll)](https://cdnjs.cloudflare.com/ajax/libs/aos/2.3.4/aos.css) library for scroll animations
- [Font Awesome](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css) for icons
- [Google Fonts](https://fonts.googleapis.com/css2?family=Roboto:wght@400;500;700&display=swap) for fonts
